# ポーカープレイヤー分析ツール

トーナメントごとにVPIP/PFRを追跡してプレイヤーのプレイスタイルを分析するWebアプリケーション

## 主な機能

### トーナメント管理
- 複数のトーナメントプロジェクトを作成・管理
- 各トーナメントごとにデータを個別保存
- トーナメント名、開催日、メモを記録

### プレイヤー分析
- 最大9人のプレイヤーを同時追跡
- VPIP/PFRの自動計算とパーセンテージ表示
- プレイヤーごとのメモ機能

### 自動分類
- **タイトアグレッシブ (TAG)**: VPIP < 25% かつ PFR > 15%
- **タイトパッシブ**: VPIP < 25% かつ PFR ≤ 15%
- **ルーズパッシブ**: VPIP ≥ 25% かつ PFR ≤ 15%
- **ルーズアグレッシブ (LAG)**: VPIP ≥ 25% かつ PFR > 15%

### カスタマイズ機能
- 分類基準の境界値を変更可能
- 最小ハンド数の設定
- トーナメントごとに異なる基準を設定可能

### その他
- データの永続化（localStorage）
- レスポンシブデザイン（スマホ対応）
- +/-ボタンで簡単に修正可能
- リアルタイム統計更新

## Vercelへのデプロイ

このプロジェクトはVercelで簡単にデプロイできます。

### 方法1: GitHub連携（推奨）

1. GitHubにこのリポジトリをpush
2. [Vercel](https://vercel.com)にアクセスしてログイン
3. "Add New Project" を選択
4. GitHubリポジトリを選択
5. デプロイ完了！

### 方法2: Vercel CLI

```bash
npm install -g vercel
cd personal-prj
vercel
```

## ローカルでの使用

ブラウザで `index.html` を開くだけで使用できます。
